# Contributing to FinAudit AI

First off, thank you for considering contributing to FinAudit AI! It's people like you that make FinAudit AI such a great tool.

## How Can I Contribute?

### Reporting Bugs
- Use a clear and descriptive title.
- Describe the exact steps which reproduce the problem.
- Explain which behavior you expected to see instead and why.

### Suggesting Enhancements
- Explain why this enhancement would be useful to most FinAudit AI users.
- Provide a step-by-step description of the suggested enhancement.

### Pull Requests
- Fill in the pull request template.
- Include screenshots and animated GIFs in your pull request whenever possible.
- Ensure the build passes.
- Follow the coding style of the project.

## Coding Style

### Backend (Python)
- Follow PEP 8.
- Use type hints wherever possible.
- Document functions and classes using docstrings.

### Frontend (JavaScript/React)
- Use functional components and hooks.
- Follow the established Tailwind CSS design system.
- Use Lucide icons for consistency.
